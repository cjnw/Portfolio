KitCrawler

KitCrawler is a Node.js web app I built to help users compare prices on athletic clothing across five major retailers in real time. Using Python-based web scrapers with BeautifulSoup, it dynamically updates a SQL database for every search. The result is a clean, responsive frontend that cuts manual price hunting by up to 90%. I designed this to give users a fast, centralized view of the best athletic gear deals online — no tabs, no hassle.

Safe Journey AI 2.0

Safe Journey AI is an Android Auto-compatible navigation app developed in Kotlin, built to help Michigan drivers choose safer routes. It presents users with three route options prioritized by safety, thanks to Mapbox SDK and a sleek Jetpack Compose UI. I also added a “Discover” feature that finds gas stations, EV chargers, and parking along the way — all while letting users plan up to 10 stops. It’s more than just directions — it’s peace of mind on the road.
